# 海龟汤智能体

这是 AgentClaw 内置 Claw Apps / 智能体广场中的官方工作流应用。它实现一个可多轮互动的海龟汤主持人：先确认玩家想玩的类型，再参考精品海龟汤生成新汤，确认汤面后进入正式游戏。

这个目录随 `agentclaw` 包发布，设计目标不是外部 demo 项目，而是 Dashboard “智能体”列表中可筛选、可运行、可查看实现、可复制改造的内置 App。

## 玩法流程

1. 开场白：介绍海龟汤规则。
2. 类型确认：询问玩家想玩悬疑、温情、反转、校园、科幻、职场等哪类汤。
3. 生成新汤：参考 `data/premium_turtle_soups.md` 的结构生成原创汤面和汤底。
4. 汤面确认：只展示汤面、类型和难度，隐藏汤底；玩家可回复“开始”“调整：更悬疑”“重新生成”。
5. 正式游戏：玩家提问、要提示或尝试作答。
6. 结算：答对后展示汤底，并总结问题数和提示数。

## 运行

```bash
agentclaw up --mode remote
```

打开 Dashboard 后进入 `智能体`，将筛选切到 `内置广场`，选择 `海龟汤主持人`。AgentClaw 服务启动时会自动注册 `agentclaw/agent_square` 下的内置工作流。

## 建议对话

```text
我想玩反转强一点的日常系海龟汤。
```

生成汤面后：

```text
开始
```

正式游戏中：

```text
死者是自愿去那个地方的吗？
给我一点提示
我猜答案是：他其实不是因为事故死亡，而是误会了某个信号。
```

## 设计约束

- 汤底不会在玩家确认前展示。
- 正式问题判定只能回复：`是`、`否`、`无关`、`部分正确`。
- 提示应引导方向，但不能直接泄露谜底。
- 玩家直接作答时，如果不完全正确，只能说明差距方向，不透露答案细节。
- 内容避免血腥、低俗、现实伤害和真实案件模仿。
